# Guessing Game Project

## Description
This project is a Bash guessing game.  
The program asks the user to guess how many files are in the current directory.
The game continues until the correct number is guessed.

## Files
- `guessinggame.sh` : Bash script for the guessing game
- `Makefile` : Runs the script
- `README.md` : Project documentation

## How to Run
1. Open a terminal
2. Run:
   ```bash
   make
